## 常见问题排查

### 问题 1：后端 API 404 错误

**症状：** 访问 `/api/v1/app/dashboard/stats` 返回 404

**原因：** 后端服务器未重启，dashboard 路由未加载

**解决：**
1. 停止后端服务器（Ctrl+C）
2. 重新启动：`python -m uvicorn backend.main:app --host 127.0.0.1 --port 8000 --reload`
3. 等待启动完成（看到 "Application startup complete"）

---

### 问题 2：前端页面空白

**症状：** 访问 `/app/dashboard/workspace` 显示空白页

**原因：** API 请求失败或数据格式不匹配

**解决：**
1. 打开浏览器开发者工具（F12）
2. 查看 Console 面板是否有错误
3. 查看 Network 面板确认 API 请求是否成功
4. 检查返回的数据格式是否与 TypeScript 类型定义一致

---

### 问题 3：菜单未显示

**症状：** 左侧导航菜单中看不到"工作台"

**原因：** 用户角色无权限或菜单未正确初始化

**解决：**
1. 检查用户角色：确保是 PM 或老板角色（有全部权限）
2. 重新运行菜单初始化脚本：
   ```bash
   cd server
   .venv/Scripts/python.exe backend/scripts/init_business_menus.py
   ```
3. 清除浏览器缓存后重新登录
4. 检查数据库：
   ```sql
   SELECT * FROM sys_menu WHERE path = '/app/dashboard/workspace';
   ```

---

### 问题 4：趋势图不显示

**症状：** 趋势图区域空白或显示错误

**原因：** `weekly_trend` 数据为空数组

**解决：**
1. 检查数据库是否有最近 7 天的反馈数据
2. 如果没有，创建一些测试数据：
   - 导入 Excel 反馈
   - 或手动录入反馈
3. 刷新页面重新加载数据

---

### 问题 5：点击列表项无跳转

**症状：** 点击紧急需求或 TOP 主题无反应

**原因：** 路由配置错误或 topic_id 不存在

**解决：**
1. 检查浏览器 Console 是否有路由错误
2. 确认 `/app/topic/detail/:id` 路由已正确配置
3. 检查 topic_id 是否有效（在数据库中存在）

## 性能验证

### API 响应时间测试

**目标：** < 500ms

**测试方法：**
```bash
# 使用 curl 测试（需要替换 TOKEN）
time curl -X GET "http://localhost:8000/api/v1/app/dashboard/stats" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

**如果超过 500ms：**
1. 检查数据库索引是否存在
2. 检查数据库连接池配置
3. 考虑添加 Redis 缓存（TTL = 1 分钟）

---

### 前端加载时间测试

**目标：** < 1 秒

**测试方法：**
1. 打开浏览器开发者工具（F12）
2. 切换到 Network 面板
3. 刷新页面
4. 查看 DOMContentLoaded 和 Load 时间

**如果超过 1 秒：**
1. 检查 API 请求是否有延迟
2. 检查前端资源加载是否有阻塞
3. 考虑添加骨架屏加载动画
