## 🔍 VOLCENGINE 聚类详情

- 希望有新反馈时能收到邮件

### 聚类 #2 (3 条反馈, 平均相似度: 0.904)

- 希望搜索能支持多个关键词，AND/OR 逻辑
- 全文搜索功能不太好用，经常搜不到明明存在的内容

### 聚类 #4 (3 条反馈, 平均相似度: 0.904)

- 希望支持多语言，我们的用户来自不同国家
- 能否添加英文界面？现在全中文不太方便
- 需要国际化支持，至少要有中英文切换

## 🎯 推荐

**人工校验要点**:

1. 查看各 provider 的聚类详情，看哪个把相似反馈聚得更准
2. 检查是否有明显不相似的反馈被错误聚在一起（误判）
3. 检查是否有明显相似的反馈没有被聚在一起（漏判）
4. 综合考虑聚类数量和质量

**切换 provider**:

```bash
# 编辑 server/backend/.env
AI_DEFAULT_PROVIDER=<provider_name>
```

**可选值**: volcengine | openai | glm | qwen
